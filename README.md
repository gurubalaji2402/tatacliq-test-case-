# tatacliq-test-case-

This project automates an end-to-end user flow on the TataCliq e-commerce website using Selenium WebDriver. The test validates product filtering, price verification, cart functionality, and UI behavior.  The automation simulates a real customer browsing watches, applying filters, validating pricing consistency, and adding a product to the cart.
